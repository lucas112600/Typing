# ⌨️ Typing...

Welcome to **Typing...**, a minimalist, Notion-inspired procedural typing trainer designed to help you focus, achieve flow state, and rigorously measure your typing speed (WPM) and accuracy. 

Out with the noisy backgrounds and bloated interfaces—**Typing...** strips everything down to pure text, system-native typography, and real-world reading material.

## ✨ Core Features

*   **Notion-Style Architecture**: A completely flat, 100% distraction-free reading experience using system-level UI fonts (`ui-sans-serif, -apple-system`) for lightning-fast performance and native aesthetics. 
*   **Real Knowledge Database**: We don't use fake repetitive words. The integrated `Database Explorer` provides you with real, curated literature excerpts (e.g., *Moby Dick*, *《三國演義》*, *Frankenstein*, *《紅樓夢》*) and cutting-edge news topics so you can read and learn while you type.
*   **Custom Parameter Tuning**: Jump in instantly via the Quick Play modes (Easy/Normal/Hard) or dial in a specific **Custom Word Count** length that matches your exact endurance training goals.
*   **Vast Bilingual Pipeline**: Natively supports full algorithm string tracking for both **Traditional Chinese** (ZH) and **English** (EN).
*   **High-fidelity Telemetry**: Features an internal real-time Local Storage Database that plots clean, high-contrast SVG graphs of your historical Words Per Minute (WPM) and overall accuracy inside the `Performance Stats` dashboard.

## 🚀 How to Use

1.  **Jump In**: Directly on the hub, choose your language (Chinese or English) and hit `Auto Start`. The system randomly pulls an article for you.
2.  **Explore**: Go to `Explore Database` (📚) to choose a specific topic like "The Apollo Program", "Semiconductor Industry", or "Philosophy of Stoicism".
3.  **Strict Mode**: Head to `Settings` (⚙️) to toggle *Strict Constraint* mode. When turned on, the cursor locks in place upon a typo and requires a perfect deletion loop, preventing bad muscle memory from forming.
4.  **Monitor Progress**: Hop into `Performance Stats` (📈) to see your lifetime moving averages and trajectory.

## 🛠 Project Stack

This application is built with peak performance in mind:
*   **Next.js 14** (App Router)
*   **React** (Server & Client ecosystem)
*   **Lucide-React** (Professional icon integrations)
*   **Pure Native CSS Variables** (No massive Tailwind overhead for thematic styling, heavily utilizing pure Notion-like UI constants).
